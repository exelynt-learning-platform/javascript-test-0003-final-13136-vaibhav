This program prints a pattern of 1s and 0s.

The outer loop runs from 1 to 5 and controls the number of rows.
The inner loop runs from 1 to i and prints the values in each row.

For every position, the program checks (i + j) % 2.

If (i + j) is even, it prints 1.
If (i + j) is odd, it prints 0.

This creates an alternating pattern of 1s and 0s.
